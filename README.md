CM1-Sakila-Database
Analisis popularitas aktor dan pola permintaan film berdasarkan wilayah pelanggan menggunakan Sakila Database dan R Markdown.

Proyek ini merupakan analisis data menggunakan Sakila Database untuk mengidentifikasi aktor dengan tingkat permintaan tertinggi serta pola permintaan film berdasarkan wilayah pelanggan. Analisis dilakukan menggunakan R Markdown, SQL, dan visualisasi data dengan ggplot2.

Tools
R
R Markdown
MySQL (Sakila Database)
ggplot2
dplyr
